# Changelog

Tüm önemli değişiklikler bu dosyada belgelenir.

## [2.0.0] - 2026-02-15

### Eklenen
- Usta/Stajyer hesap sistemi
- Bildirim sistemi (eksik evrak, istek)
- Gece/Gündüz tema desteği
- Landing page (tanıtım sayfası)
- Versiyon yönetim sistemi
- API dokümantasyonu (Swagger)
- Mobil uygulama API desteği

### Değiştirilen
- Tarih bazlı kayıt sıralaması güncellendi
- API endpoint'leri optimize edildi

## [1.1.0] - 2026-02-15

### Eklenen
- 5 şube sistemi (Bursa, İzmit, Orhanlı, Hadımköy, Keşan)
- Personel yönetimi (Garanti/Hasar Danışmanı, Müşteri Kabul)
- İş emri numarasından şube algılama
- Şube bazlı veri filtreleme
- WhatsApp kısayolu
- Admin dashboard (şube genel durumu)
- Kullanıcı menüsü ve logout

### Değiştirilen
- Staff kullanıcıları sadece kendi şubelerinin verilerini görebilir
- Dashboard şube bazlı istatistikler eklenmiş

## [1.0.0] - 2026-02-15

### Eklenen
- İlk sürüm
- 4 kayıt türü (Standard, RoadAssist, Damaged, PDI)
- WhatsApp benzeri kayıt arayüzü
- Fotoğraf, Video, PDF yükleme
- Plaka OCR (Tesseract.js)
- Sesli not alma (Web Speech API)
- Offline queue desteği (IndexedDB)
- Türkçe/İngilizce dil desteği
- Admin paneli (Dashboard, Kayıtlar, Ayarlar)
- JWT kimlik doğrulama
